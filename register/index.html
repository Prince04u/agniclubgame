Eligibility

RequirementDetails
Age18+
MobileOTP capable number
PasswordStrong unique password

Registration Steps

  1. Open the official website.
  2. Select Register.
  3. Enter your mobile number.
  4. Verify OTP.
  5. Create a strong password.
  6. Add an invite code if available.
  7. Accept terms.
  8. Create your account.

Benefits

  • Secure login
  • Profile management
  • Wallet access
  • Transaction history
  • Reward notifications

Common Mistakes

  • Wrong mobile number
  • Weak password
  • Sharing OTP
  • Using unofficial websites

Security Tips

  • Never share OTP.
  • Use unique passwords.
  • Avoid public devices.
  • Review account activity regularly.

Troubleshooting

OTP Not Received

Wait a few minutes and request a new OTP.

Registration Failed

Check your details and internet connection.
