Agni Club Sign Up Walkthrough

If you've never created an Agni Club account before, this short walkthrough explains the sign-up screens in plain language so you can complete onboarding in a couple of minutes.

Why Sign Up?

A registered account lets you save your activity history, claim welcome bonuses where available, and use the wallet, gift code and referral features.

The Sign-Up Screen

The sign-up screen typically asks for a mobile number, a password, and an optional invite code. Some builds also include an email field for password recovery — this is optional but recommended.

Field-by-Field Tips

  • Mobile number: Use a number you actively use — you'll receive OTPs and recovery messages here.
  • Password: Long passphrases beat short complex ones. Aim for 12+ characters.
  • Invite code: Optional. Affects the referral bonus, nothing else.

After You Finish Sign Up

You'll land on the home dashboard. Take a minute to enable any two-step verification option offered and review the responsible-play settings.

Frequently Asked Questions

Is sign up the same as registration?

Yes — they're two names for the same flow.

Can I change my number later?

Some apps allow it through support verification — check the help center for the latest policy.
