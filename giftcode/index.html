Agni Club Gift Code & Promo Code Guide

Gift codes are short promotional strings that occasionally unlock a small bonus inside your in-app wallet. This guide explains how they work and how to redeem them safely.

What is a Gift Code?

A gift code (also called a promo code, redeem code or reward code) is a short alphanumeric string distributed during events, milestones or partner campaigns. When entered in the official Gift Code section, a valid code adds a small bonus to your account wallet.

How to Redeem a Gift Code

  1. Sign in to your Agni Club account.
  2. Open the Gift Code or Promo Code section from your dashboard.
  3. Enter the exact code — case sensitivity may apply.
  4. Tap Redeem.
  5. If valid, the bonus will reflect in your wallet immediately.

Where Real Codes Are Shared

Official codes are usually shared through the in-app announcements, the official website's news section, and verified community channels. Treat anything else with caution.

Watch Out for Fake Codes

Heads up. Random codes posted by strangers asking you to share OTPs, install third-party APKs, or pay an unlock fee are scams. A genuine code never asks for any of these.

Common Reasons a Code Fails

  • The code has expired.
  • The code is region-limited and your account is outside that region.
  • You've already redeemed it on this account.
  • You mistyped a character — try again carefully.

Frequently Asked Questions

Can I get a gift code by signing up?

Sometimes a welcome event offers a starter code — check the active offers screen after you register.

How many gift codes can I redeem per day?

Limits vary by event. Each code is usually one-use per account.

Do codes have expiry dates?

Yes — almost every promotional code has a short validity window.
