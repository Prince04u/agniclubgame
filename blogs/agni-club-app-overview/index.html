Agni Club App Overview — Features You Should Know

A friendly tour of the {BRAND} app: the home dashboard, mini-games library, wallet section, and the settings every new user should review on day one.

First Impressions

The first thing new users notice in Agni Club is how compact the layout is. The bottom navigation keeps the core sections — Home, Games, Wallet, Rewards, Profile — exactly one tap away. There are no nested menus to fight with.

The Mini-Games Library

Each game card shows the round timer, the entry option and a brief rules tooltip. Casual players can browse by category, while returning users can jump straight into a familiar favourite.

Wallet Section

The wallet section handles deposits, withdrawals and history. It's worth spending two minutes scrolling through it before your first session so you know exactly where every action lives.

Settings Worth Reviewing

  • Enable any two-step verification that's offered.
  • Review notification preferences — quiet hours help limit distraction.
  • Set a personal session reminder if the option exists.
